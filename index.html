Introdução

Conteúdo da documentação...

Download

Baixe os arquivos CSS e JS compilados do design system para usar em seus projetos.

Download Completo

Baixe a pasta completa dist como ui-kit.zip contendo todos os arquivos CSS e JS compilados.

CSS

Arquivo CSS minificado compilado do design system.

Baixar bootstrap.min.css

Baixar bootstrap.min.css.map

JavaScript

Arquivo JavaScript minificado do Bootstrap.

Baixar bootstrap.bundle.min.js

Instalação

Após baixar os arquivos, inclua-os no seu projeto HTML:

<!-- CSS -->
<link href="css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript -->
<script src="js/bootstrap.bundle.min.js"></script>
